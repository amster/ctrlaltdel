CtrlAltDel

A simple utility to add/manipulate/delete objects in a frame. Requires jQuery.

Demo

Add an element, move it around, remove it.

Usage

Create a DIV and initialize a CtrlAltDel object. Make sure the DIV is is not position:static. 


<div id="screen"></div>
<script>
  var $target = $('#screen'),
      screen = new SEQ.CtrlAltDel( $target );
</script>

Issue a command to add/modify an object. Make sure the "id" is unique—this is what will be used to see if an element is created or just updated.

If the "bubble" does not exist then a DIV will be added to $target ided "bubble" with inner HTML html, CSS classes classes, and css. 


<script>
  var $element_updated = screen.set('bubble', {
    classes: 'soapy floaty',
    html: 'Some <b>html',
    css: {
      top: 10,
      left: 10,
      backgroundColor: 'red'
    }
  });
</script>

Updating this ID means just setting the CSS. The HTML will only be updated if it is different than it was last time.

Getting rid of the element is super easy. Just use remove: 


<script>
  screen.remove({
    id: 'bubble'
  });
</script>

Have fun!
